# 📊 Netflix Data Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an **interactive Netflix Dashboard built using Power BI** to analyze movies and TV shows available on Netflix.
The dashboard provides insights into **content distribution, top actors, top directors, yearly growth, and overall platform statistics**.

The goal of this project is to demonstrate **data analysis, data visualization, and business intelligence skills using Power BI**.

---

## 🎯 Objectives

* Analyze the distribution of **Movies vs TV Shows**
* Identify **Top Actors and Directors**
* Understand **Yearly content growth on Netflix**
* Calculate **Average ratings and total runtime**
* Provide **interactive filters for better exploration**

---

## 📂 Dataset

The dataset used in this project contains information about Netflix titles including:

* Title
* Type (Movie / Show)
* Cast
* Director
* Release Year
* Runtime
* IMDB Rating
* TMDB Rating

### Files Used

* `titles.csv`
* `credits.csv`

## 📊 Dashboard Features

### 1️⃣ Content Distribution

Shows the percentage of:

* **Movies: 63.31%**
* **TV Shows: 36.69%**

---

### 2️⃣ Top Actors

Displays actors who appeared most frequently in Netflix titles.

Examples:

* Boman Irani
* Kareena Kapoor Khan
* Takahiro Sakurai
* Priyanka Chopra

---

### 3️⃣ Top Directors

Highlights directors with the highest number of titles.

Examples:

* Raúl Campos
* Jan Suter
* Ryan Polito
* Jay Karas

---

### 4️⃣ Yearly Progression

Visualizes how the number of Netflix titles has **grown over the years**, especially after **2020**.

---

### 5️⃣ Key Performance Metrics

The dashboard includes important KPIs:

| Metric          | Value       |
| --------------- | ----------- |
| Total Titles    | 5645        |
| Total Runtime   | 7.15K Hours |
| Avg IMDB Rating | 6.50        |
| Avg TMDB Rating | 6.83        |
